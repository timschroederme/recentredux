Recent Redux Help

Getting Started

Recent Redux is a tool keeping track of files and folders ("items") which have been recently used on your computer. It is updated live by using Apple's Spotlight service and allows for direct access to these items. If you want, you can also define filters to show you just those items you need.

Main Window

What you see in the application's main window are items on your local storage media which have been recently accessed. They are sorted by date and time with the most recent ones being at the top of the list. As they are accessed or modified while Recent Redux is running, the list is updated live.

Filter Bar

Above the list of recently accessed items is the Filter Bar (see also Filter Preferences). It shows all filters not disabled in the Filter Preferences. You can click on a filter tag to activate this filter, click again to deactivate it. The Main Window and the Dock Menu will show only recently accessed items matching the filter criteria. You can select more than one filter at the same time by holding the <Shift> key while clicking on a filter tag. In case the Main Window is not wide enough to display all filter tags, the remaining filter tags will be displayed in an overflow context menu accessible by clicking on the arrow symbol at the right side of the Filter Bar.

Hotkey and Always in Front

If you have the Global Hotkey enabled (see General Preferences), you can bring the main window to front by simply clicking the <alt> + <cursor left> keys at the same time. If you want to have the main window stay at front all the time, you can do so by checking the Always in Front Option in the Windows Menu.

Dock Menu

Like the main window, the dock menu (secondary click on the application's dock icon while it is running) shows the ten most recent accessed items. The dock menu is not updated live.

What Do You See?

What you see in the main window (and in the dock menu) is the result of raw Spotlight data returning normally an awful lot of information about items accessed on your computer. This huge amount of information is then filtered two times:
  1. First, according to the settings you have provided in the Preferences (see Search Preferences), Spotlight results are narrowed in time, location and type of access.
  2. Second, the results are further narrowed down by the filters you have selected in the Filter Bar or in the Filter Menu (see Filter Preferences).

Actions

You can perform a number of actions on the items shown in the main window. These actions are similar to actions available in Finder. You can use either Recent Redux' Main Menu to perform an action or use the Context Menu of the Main Window (secondary click on an item in the Main Window). See Actions for a detailed description of what actions are available.
