Recent-Redux-Hilfe

Aktionen

Mit Recent Redux können Sie eine Reihe von Aktionen mit den Dateien und Ordnern durchführen, die im Hauptfenster oder im Dock-Menü angezeigt werden. Die verfügbaren Aktionen können über das Hauptmenü oder über das Kontextmenü des Hauptfensters (Sekundär-Klick auf das Hauptfenster) aufgerufen werden. Die meisten Aktionen sind im Dock-Menü jedoch nicht verfügbar.

Recent Redux unterstützt die folgenden Aktionen:

  • Vorschau: Sie können eine Vorschau des Inhalts der Datei bzw. des Ordners anzeigen lassen, dabei verwendet Recent Redux Apples Quick Look. Per Tastatur ist die Vorschau mit der <Space>-Taste verfügbar.

  • Öffnen: Sie können den ausgewählten Eintrag mit seiner Standard-Anwendung öffnen. Dies geschieht auch, indem Sie im Hauptfenster einen Doppelklick auf den Eintrag ausführen oder im Dock-Menü einfach auf einen Eintrag klicken.

  • Öffnen mit: Sie können den ausgewählten Eintrag mit einer beliebigen auf ihrem Rechner installierten Anwendung öffnen; dabei wird Ihnen Recent Redux einige Vorschläge für geeignete Anwendungen machen. Diese Aktion ist für Ordner und für Anwendungen nicht verfügbar.

  • Kopieren: Sie können die ausgewählte Datei bzw. den ausgewählten Ordner in die Zwischenablage kopieren.

  • Pfad kopieren: Sie können den Pfad des ausgewählten Eintrags in die Zwischenablage kopieren.

  • In Finder anzeigen: Diese Aktion öffnet ein Finder-Fenster, in dem der Speicherort des ausgewählten Eintrags angezeigt wird.

  • Per Email verbreiten: Mit dieser Aktion können Sie die ausgewählte Datei bzw. den ausgewählten Ordner per Email versenden. Recent Redux öffnet dazu eines der auf ihrem Rechner installierten Email-Programme, wenn möglich, Ihr als Standard-Programm eingestelltes Email-Programm. Derzeit unterstützt Recent Redux Apples Mail-Programm, Microsoft Entourage 2008, Microsoft Outlook 2011 und Postbox. Recent Redux wird das Email-Programm starten bzw. in den Vordergrund bringen, eine neue Email-Nachricht erstellen und die ausgewählte Datei bzw. den ausgewählten Ordner als Anhang anfügen. Sie können sodann die Nachricht vervollständigen und absenden. Wenn ihr als Standard-Programm ausgewähltes Email-Programm von Recent Redux nicht unterstützt wird, wird Recent Redux Sie zur Auswahl eines unterstützten Email-Programms aufforden. Ihre Wahl wird gespeichert und kann in den Einstellungen geändert werden (siehe Allgemeine Einstellungen). Die Größe von Emails ist derzeit auf 25 Megabytes begrenzt.
