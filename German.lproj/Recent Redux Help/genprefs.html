Recent-Redux-Hilfe

Allgemeine Einstellungen

In den Allgemeinen Einstellungen von Recent Redux können Sie einige allgemeine Parameter festlegen.

Hotkey

Sie können die Hotkey ein- oder ausschalten. Zur Zeit ist es nicht möglich, selbst festzulegen, welche Taste als Hotkey verwendet wird.

Standard-Email-Client

Sie können auswählen, welcher der auf Ihrem Computer installierten und von Recent Redux unterstützten Email-Clients von Recent Redux zum Versenden von Emails verwendet werden soll (siehe Aktionen).
